# AnyShop API

## Pre-requisitos

- Docker Desktop

## Correr el proyecto

1. Clonar el repositorio

```bash
git clone
```

2. Instalar las dependencias del proyecto

```bash
npm install
```

3. Renombrar el archivo **.env.example** a **.env** y editar los valores

4. Levantantar la base de datos

```bash
docker-compose up -d
```
